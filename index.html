Icon Cards

Os Icon Cards tem por objetivo apresentar um cartão ou um conjunto de cartões. Estes cartões podem ter cores, tamanhos e formatos diferentes. Outro ponto importante é que o conteúdo dos cards podem ser letras ou também ícones, como mostram os exemplos abaixo.

Os cartões podem ser usados como links <a> ou como blocos <div>. Para usar como link, use a tag HTML <a href="#" class="icon-card"></a>

Icon Cards - Sombras

Para usar os cartões com sombra no texto (text-shadow) é necessário declarar a cor do cartão ao mesmo tempo ( exemplo: class="icon-card branco text-shadow"). Nos demais tipos de sombra não há dependência com a cor do cartão, pois são todas variações do preto.

Sombra no texto
Y
Sombra no cartão
A
E
Blur no texto
Z
Blur no cartão
Exemplos 
                
<div class="icon-card branco text-shadow">...</div>
<div class="icon-card shadow">...</div>
<div class="icon-card text-blur">...</div>
<div class="icon-card blur">...</div>
